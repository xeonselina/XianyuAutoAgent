# 2026-03-03 工作记录

## 修复闲鱼客服消息处理问题

### 问题1：WebSocket 消息重复处理
- 闲鱼 sync 协议会偶尔重复推送同一条消息，导致 Agent API 被调用两次
- 在 `message_handler.py` 中新增 `MessageDeduplicator` 类，基于 `message_id + chat_id` 做 30 秒窗口去重
- 去重检查在数据库日志写入之前执行

### 问题2：数据库保存失败 - 线程安全
- 报错：`'NoneType' object has no attribute 'settimeout'`
- 原因：`pymysql.Connection` 不是线程安全的，`asyncio.to_thread` 并发调用 `save_message` 时产生竞争
- 在 `conversation_store.py` 中加入 `threading.Lock()` 保护 `save_message` 方法
