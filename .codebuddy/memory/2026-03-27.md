## Eval error 误标问题修复 (87/765 → 应该是 0)
- **根因**: confidence guard 把低置信度回复清空为 `""`, 但 metadata 中的 `confidence_percent`/`response_suppressed` 没有从 turn → executor → eval_replay 传递出来
- **链路**: turn.py 的 TurnResult.metadata 缺字段 → executor.py 返回的 metadata 只有 duration_ms → eval_replay.py 用 `bool(response)` 判定 status，空字符串 = error
- **修复 1**: `turn.py` TurnResult.metadata 加入 confidence_percent, response_suppressed
- **修复 2**: `executor.py` run() 返回值 metadata 传递最后一个 turn 的 confidence 信息
- **修复 3**: `eval_replay.py` status 判定改为三分支：response_suppressed→success, has_error→error, 其他→success
