Q: On tool-continuation turns (is_tool_continue=True), what system prompt is 
   sent? Is the full long system prompt re-sent on every tool-continuation turn?

A: YES - The FULL system prompt is RE-SENT on every tool-continuation turn.

   ✓ System prompt loaded fresh (or from 5-min cache) each turn
   ✓ System prompt is IDENTICAL across all turns within a run()
   ✓ System prompt includes injected item_info & context_summary slots
   ✓ System prompt is NOT omitted on tool-continuation
   ✓ Qwen API receives full system prompt on EVERY turn
   ✓ All messages (including system prompt) are persisted to agent_turns.llm_input
