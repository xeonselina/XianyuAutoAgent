# 变更：增加物流时效不匹配警告

## 为什么
工作人员新建或调整租赁档期时，手动选择的物流天数可能少于顺丰标快送达目的地的预估时效。如果未注意到这一差异就直接保存，设备可能无法在约定日期前送达。

## 变更内容
- 通过库存管理系统的物流预估接口复用现有的深圳始发顺丰标快时效规则。
- 新建或更新租赁记录前，将所选物流天数与目的地预估时效进行比较。
- 当所选天数少于预估天数时显示橙色警告，同时允许工作人员明确确认后继续。
- PC 端和移动端的新建、编辑流程均实现相同行为。
- 为时效预估和警告判断补充自动化测试。

## 影响范围
- 受影响的规格：`rental-logistics-warning`
- 受影响的代码：库存管理系统租赁 API 和工具、PC 端预定及编辑弹窗、移动端新建及编辑页面、相关测试
- 不涉及数据库或迁移变更
