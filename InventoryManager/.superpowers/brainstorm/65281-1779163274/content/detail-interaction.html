A · 底部抽屉（Bottom Sheet）

点击租赁条后，从底部滑出一个抽屉面板，展示完整租赁信息和操作按钮。背景甘特图变暗。下滑或点击背景关闭。

优点

  • iOS 原生感强
  • 操作区域大，拇指易及
  • 可展示完整信息

缺点

  • 遮挡甘特图

B · 行内展开

点击某设备行后，该行向下展开，在甘特图内嵌入详情区域。甘特图其他行正常显示，不遮挡。再次点击折叠。

优点

  • 不遮挡其他设备
  • 上下文清晰

缺点

  • 展开后需要滚动
  • 操作按钮偏小

C · 跳转详情页

点击租赁条跳转到独立的详情页面，信息展示最完整，操作按钮最大。点击返回回到甘特图，会记住滚动位置。

优点

  • 信息展示最完整
  • 操作区域大
  • 易于扩展更多操作

缺点

  • 来回跳转，效率稍低
