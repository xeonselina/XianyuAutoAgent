# 身份证点击放大功能说明

## 功能概述
在租赁合同页面，用户上传身份证照片后，可以点击身份证图片来放大查看，提供更好的用户体验。

## 新增功能

### 1. 图片预览对话框组件 (ImagePreviewDialog.vue)
- 位置：`src/components/ImagePreviewDialog.vue`
- 功能：
  - 显示放大的身份证图片
  - 支持关闭对话框
  - 提供图片下载功能
  - 响应式设计，适配不同屏幕尺寸

### 2. 租赁合同页面增强
- 位置：`src/views/RentalContractView.vue`
- 新增功能：
  - 身份证图片添加点击事件
  - 鼠标悬停效果（缩放和阴影）
  - 点击后打开图片预览对话框

## 使用方法

1. **上传身份证**：在租赁合同页面上传身份证照片
2. **点击查看**：点击已上传的身份证图片
3. **放大预览**：图片会在对话框中放大显示
4. **下载图片**：可以在预览对话框中下载身份证图片
5. **关闭预览**：点击关闭按钮或按ESC键关闭对话框

## 技术实现

### 组件结构
```
ImagePreviewDialog.vue
├── 对话框容器 (el-dialog)
├── 图片显示区域
├── 底部操作按钮
└── 响应式样式
```

### 主要特性
- 使用Vue 3 Composition API
- Element Plus UI组件库
- TypeScript类型支持
- 响应式设计
- 图片加载错误处理
- 下载功能支持

### 样式特性
- 鼠标悬停效果
- 平滑过渡动画
- 移动端适配
- 图片居中显示
- 阴影和圆角设计
